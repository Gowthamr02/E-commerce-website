# React E-Commerce Web Application

A responsive e-commerce web application built using **React.js**.
The application allows users to browse products, view product details, and simulate cart and checkout functionality.

## 🚀 Live Demo

https://shopify-app-liart.vercel.app

## 🛠 Tech Stack

* React.js
* JavaScript
* Context API
* Axios
* HTML5
* CSS3
* React Router

## ✨ Features

* Product categories (Fashion, Electronics, Accessories)
* Product detail pages
* Shopping cart functionality
* Checkout simulation
* Responsive UI design
* Global state management using Context API

## 📦 Installation

Clone the repository

```
git clone https://github.com/Gowthamr02/E-commerce-website
```

Install dependencies

```
npm install
```

Run the project

```
npm run dev
```
